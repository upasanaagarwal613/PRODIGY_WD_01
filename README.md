# PRODIGY_WD_01 – Interactive Navigation Menu

## 📌 Overview
This project is an interactive and responsive navigation menu built using HTML, CSS, and JavaScript.

The navigation bar remains fixed at the top of the page, changes style on scroll, and includes hover effects for menu items.

## ✨ Features
- Fixed navigation bar
- Scroll-based background change
- Hover animation effects
- Active link highlighting
- Smooth scrolling
- Responsive design with hamburger menu
- Clean and modern UI

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript
